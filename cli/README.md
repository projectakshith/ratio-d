## what is this

ratio'd cli is a terminal interface for SRM's academia portal. same data as the web dashboard ‚ÄĒ attendance, marks, timetable, bunk calculator ‚ÄĒ but you never open a browser.

built with **Go** and **[Bubble Tea](https://github.com/charmbracelet/bubbletea)** for a proper interactive TUI.

---

## planned stack

| layer | tech |
|---|---|
| language | Go |
| tui framework | [Bubble Tea](https://github.com/charmbracelet/bubbletea) (charmbracelet) |
| styling | [Lip Gloss](https://github.com/charmbracelet/lipgloss) |
| http | Go standard `net/http` + the ratio'd backend API |
| auth | same AES-256 credential flow as the web app |
| landing page | Next.js (this repo) |

## planned features

- `ratio-d login` ‚ÄĒ authenticate once, credentials stored locally
- `ratio-d attendance` ‚ÄĒ view attendance with bunk calculator
- `ratio-d marks` ‚ÄĒ current internal marks + target calculator
- `ratio-d timetable` ‚ÄĒ today's / weekly schedule
- `ratio-d alerts` ‚ÄĒ configurable terminal notifications
